이슈 API

이슈 저장 API

Request

POST /api/issues HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Content-Length: 133
Host: api.rarus-be.com

{
  "title" : "Issue Title",
  "content" : "Issue Content",
  "assigneeIds" : [ 1 ],
  "labelIds" : [ 1 ],
  "milestoneIds" : [ 1 ]
}

Request Body

{
  "title" : "Issue Title",
  "content" : "Issue Content",
  "assigneeIds" : [ 1 ],
  "labelIds" : [ 1 ],
  "milestoneIds" : [ 1 ]
}

Request Fields

Path Type Description

title

String

이슈 제목

content

String

이슈 내용

assigneeIds

Array

이슈 담당자 아이디

labelIds

Array

레이블 아이디

milestoneIds

Array

마일스톤 아이디

Response

HTTP/1.1 200 OK
Content-Type: application/json;charset=UTF-8
Content-Length: 1

1

Response Fields

Value Description

1

저장된 이슈 아이디

Curl And Httpie

Curl
$ curl 'https://api.rarus-be.com/api/issues' -i -X POST \
    -H 'Content-Type: application/json;charset=UTF-8' \
    -H 'Accept: application/json' \
    -d '{
  "title" : "Issue Title",
  "content" : "Issue Content",
  "assigneeIds" : [ 1 ],
  "labelIds" : [ 1 ],
  "milestoneIds" : [ 1 ]
}'
Httpie
$ echo '{
  "title" : "Issue Title",
  "content" : "Issue Content",
  "assigneeIds" : [ 1 ],
  "labelIds" : [ 1 ],
  "milestoneIds" : [ 1 ]
}' | http POST 'https://api.rarus-be.com/api/issues' \
    'Content-Type:application/json;charset=UTF-8' \
    'Accept:application/json'
